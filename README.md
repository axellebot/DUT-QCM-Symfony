IUT-MP-PHP
==========

A Symfony project created by MarvinLockwood, axellebot.

#Install
##For devs
* clone the project
* install composer
* run "composer install"
* create "symfony" database 
* generate all table with Doctrine : "php bin/console doctrine:schema:update --force"
